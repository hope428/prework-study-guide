HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Section elements create separate sections for each page
  • UL elements create an unordered list, LI elements contain each individual list item

CSS

  • CSS Adds styling to a page
  • You can select classes with the . operator
  • You can style entire elements by selecting them
  • ID's can be selected with the # operator
